# QuickBill - Simple Invoice Generator

QuickBill is a simple, fast, and clean digital billing and invoice generator for small businesses. It allows you to create professional invoices in seconds without any signup required.

## Features

- Lightning-fast invoice generation
- Professional invoice templates
- Mobile-friendly responsive design
- PDF download functionality
- Auto-calculation of totals
- No backend required (pure HTML, CSS, JavaScript)

## How to Deploy on Netlify

1. **Prepare your files**: Create a folder with the following files:
   - `index.html`
   - `style.css`
   - `script.js`

2. **Create a Netlify account** (if you don't have one):
   - Go to [netlify.com](https://netlify.com)
   - Sign up for a free account

3. **Deploy using Drag & Drop**:
   - Log in to your Netlify account
   - Go to the Sites page
   - Drag and drop your project folder onto the deployment area
   - Netlify will automatically deploy your site

4. **Alternative: Deploy using Git**:
   - Push your code to a Git repository (GitHub, GitLab, etc.)
   - Connect your repository to Netlify
   - Netlify will automatically deploy when you push changes

## Customization

You can customize QuickBill by:

- Changing colors in the CSS `:root` variables
- Modifying the invoice template in the HTML
- Adding new fields to the form
- Adjusting the PDF generation options

## Browser Compatibility

QuickBill works in all modern browsers including:
- Chrome
- Firefox
- Safari
- Edge
